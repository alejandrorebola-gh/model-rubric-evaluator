# Model Rubric Evaluator

A browser-based tool for comparing GPT, Gemini, and Claude against weighted evaluation rubrics.

## Features

- Paste rubric criteria directly from Mantis
- Automatically extract criterion weights
- Evaluate GPT, Gemini, and Claude side by side
- Compute weighted totals
- Compute normalized percentages
- Support positive and negative criteria
- Notes column for reviewer comments

## Usage

1. Paste the rubric criteria.
2. Click "Generate Table".
3. Enter 0 or 1 for each model and criterion.
4. Click "Recalculate Totals".
5. Review weighted scores and percentages.

All calculations run locally in the browser.
